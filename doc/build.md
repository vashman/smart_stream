Building
==========================================================================
Compile all files in [src/](../src/) directory, and link manually. Or use
a build script in the [src/build](../src/build) directory.

example g++:
`g++ -std=c++11 -c *.cpp`

example clang++:
`clang++ -std=c++11 -c *.cpp`
